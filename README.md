## Structure
- `command.sheet.sh`: command sheet to install, enable, disable, start, restart, ...
- `conf`: nginx configuration files
- `rendu`: folder of files to be returned for the project
- `front`: front-end code (React, TS)
- `back`: back-end code (Express, TS)
